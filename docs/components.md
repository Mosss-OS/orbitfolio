# Components

Detailed documentation of the UI components in Orbitfolio.

## Core Components

### Navbar
Navigation bar with responsive design and mobile menu support.

### HeroSection
The main landing section with headline, description, and CTA buttons.

### Globe3D
Interactive 3D globe visualization using Three.js.

```tsx
import Globe3D from './components/Globe3D';

function App() {
  return <Globe3D />;
}
```

### StatsBar
Displays statistics using Recharts for data visualization.

### HowItWorks
Explains the project functionality with step-by-step cards.

### FeaturesSection
Grid layout showcasing key features.

### CTASection
Call-to-action section encouraging user engagement.

### Footer
Site footer with links and copyright information.

## UI Components

The project includes a comprehensive set of UI components built with Radix UI and styled with Tailwind CSS:

- Button
- Card
- Dialog
- Dropdown Menu
- Input
- Select
- Toast
- And many more...

See `src/components/ui/` for the complete list.
